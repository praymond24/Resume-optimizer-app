## How It Works

1. **Upload Your Resume**: Either upload your current resume file or paste the text
2. **Add Job Description**: Upload or paste the job description you're applying for
3. **Generate**: Click the optimize button and let AI analyze and improve your resume
4. **Copy & Use**: Review the optimized resume and copy it to use in your applications

## Local Development

Simply open `index.html` in your web browser. No build process or dependencies required!

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/resume-optimizer-app.git

# Navigate to the directory
cd resume-optimizer-app

# Open in browser
open index.html
```
